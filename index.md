## 代表项目

### GenAI Proxy
基于 Rust 的代理服务器，将 SAP AI Core 暴露为标准 AI API（兼容 OpenAI / Anthropic / Gemini），使团队可使用主流 AI 工具接入 SAP 合规 AI 平台。

### TalentTunnel
Go 语言实现的内网穿透服务（类 ngrok），支持 WebSocket 长连接、Auth Key 认证、实时 Dashboard 管理界面和完整数据统计。

### VoiceBridge
macOS 实时 AI 语音翻译应用，打破会议中的语言障碍。

### AudioBookOfPlex
SwiftUI 原生 iOS 有声书播放器，集成 Plex Media Server。

### ApplicationPad
macOS 原生应用启动器，提供流畅的应用管理体验。

### STTInput
轻量级语音辅助输入工具，通过语音识别将语音转为文字并输入到任意应用。

### RBA Stats
篮球爱好者数据统计 Web 应用，支持评分、AI 分组、排行榜功能。

### 五行弈（WuxingDeck）
Unity 开发的 Roguelike Deckbuilder 卡牌游戏（类杀戮尖塔）。
